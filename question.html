Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML HTML tags soho content dekhay, innerText sudhu visible text dekhay, ar textContent shob text hidden o dekhay, HTML tags ke ignore kore.

Question-3: What is event delegation in the DOM?

Event delegation holo ekta technique, jekhane ekta parent element e single event listener add kora hoy, ar oi listener ta tar child elements er events handle kore.

Question-4: What is event bubbling in the DOM?

Event bubbling hocche, jokhon kono event child element e hoy, seta parent elements e "bubble up" kore. Ete parent elements child elements er events handle korte pare.
